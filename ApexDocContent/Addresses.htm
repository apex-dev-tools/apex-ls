Address Management Related Classes

Address Management in NPSP is made up of several components.

  • Multiple Address objects supported on Accounts (both Household and Organization Accounts)
  • Seasonal Address support
  • Address Validation Services such as Smarty Streets, Cicero, and Google Maps

The classes involved with Address object support include the following trigger handlers:

  • ADDR_Account_TDTM
  • ADDR_Addresses_TDTM
  • ADDR_Contact_TDTM

The classes involved with Seasonal Address support include the following schedulable class

  • ADDR_Seasonal_SCHED

The classes involved with Address Verification support include:

  • ADDR_Cicero_Validator
  • ADDR_GoogleGeoAPI_Validator
  • ADDR_IValidator
  • ADDR_SmartyStreets_Gateway
  • ADDR_SmartyStreets_Validator
  • ADDR_Validator_Batch
  • ADDR_Validator_REST
  • ADDR_Validator_TDTM
  • ADDR_Validator
