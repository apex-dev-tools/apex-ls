Payment Classes

NPSP Payments are optional child objects under Opportunities, used to track payments against the Opp, as well as writeoffs.

The classes involved with Payments include:

  • PMT_Payment_TDTM - the Trigger Handler on Opportunities that manages Payments on the Opportunity
  • PMT_PaymentCreator - the class that does all the work for the Trigger Handler
  • PMT_PaymentCreator_TEST - test code
  • PMT_PaymentWizard_CTRL - the visualforce page Controller for both the Schedule Payments and Writeoff Payments wizards
  • PMT_PaymentWizard_TEST - test code
