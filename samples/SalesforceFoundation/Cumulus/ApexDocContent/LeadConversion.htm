Lead Conversion Related Classes

Lead Conversion consists of a Visualforce page controller and its associated test class. The controller provides all logic for converting leads

Go to Setup | Customize | Leads | Buttons, Actions and Links and select 'Edit' next to 'Convert' to set the override for the Visualforce page

  • Overrides standard conversion behavior to better support default NPSP Account models
  • Handles 1x1, Household and Individual Account models
  • Using 'self' or 'individual' as the Company name will trigger the Lead to be created as a 1x1 or Individual Account

All logic is contained in LD_LeadConvertOverride_CTRL
