Address Management Related Classes

Address Management is made up of several components:

  • Multiple Address children supported on Account and Contact
  • Seasonal Address support

Classes involved with Address object support:

  • ADDR_Account_TDTM - Trigger Handler on Account for Address management. Creates Address record from Account.
  • ADDR_Addresses_TDTM - Trigger handler on Address object to support propagating address changes.
  • ADDR_Addresses_UTIL - Utility methods for address-related functionality.
  • ADDR_Contact_TDTM - Trigger Handler on Contact for Address management. Creates Address record from Contact.

Classes involved with Seasonal Address support:

  • ADDR_Seasonal_BATCH - Updates seasonal addresses in batch.
  • ADDR_Seasonal_SCHED - Supports the automatic scheduling of Seasonal Address updates.
  • ADDR_Seasonal_UTIL - Utility methods for seasonal address-related functionality.
