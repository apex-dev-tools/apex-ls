Utilities Classes

These classes are reusable generic utilities.

  • UTIL_CustomSettingsFacade - Helper class that retrieves custom settings and sets defaults.
  • UTIL_Debug - Utility class for debug statements to turn them off in a managed context, as well as to display relevant performance information.
  • UTIL_Describe - Allows easy access to object and field description information from within other classes.
  • UTIL_Namespace - Holds static methods for common utilities, such as detecting whether we are in a managed package, and some string helpers.
