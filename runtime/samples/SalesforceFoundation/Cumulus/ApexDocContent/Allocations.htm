Allocations Related Classes

Allocations in NPSP is made up of several components.

  • Allocations trigger support
  • Manage Allocations page
  • Allocations rollups
  • Make Default Allocations feature
  • Multi currency support

The classes involved with Allocations trigger support include:

  • ALLO_Allocations_TDTM
  • ADDR_Addresses_TDTM
  • ADDR_Contact_TDTM

The classes involved with the Manage Allocations page include:

  • ALLO_ManageAllocations_CTRL
  • ALLO_ManageAllocations_TEST

The classes involved with Allocations rollups include:

  • ALLO_Rollup_SCHED
  • UTIL_AbstractRollup_BATCH
  • ALLO_RollupBTN_CTRL
  • ALLO_Rollup_TEST
  • ADDR_SmartyStreets_Validator
  • ADDR_Validator_Batch
  • ADDR_Validator_REST
  • ADDR_Validator_TDTM
  • ADDR_Validator

The classes involved with the Make Default Allocations feature include:

  • ALLO_MakeDefaultAllocations_BATCH

The classes involved with multi currency support include:

  • ALLO_Multicurrency_TDTM
